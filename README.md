## keyboard control

- <kbd>Esc</kbd> quit
- <kbd>R</kbd> toggle debugging boundary rectangle overlay
- <kbd>T</kbd> toggle debugging rotating test text
- <kbd>A</kbd> toggle animation on/off
- <kbd>C</kbd> open control window
